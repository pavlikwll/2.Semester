# Unity Final Project

A small 2D top-down adventure game created in Unity as the final assignment for the Unity course at Kunstschule Wandsbek.

## Features

- 2D Top-Down gameplay
- Player movement
- Roll mechanic
- Sword combat
- NPC interactions
- Dialogue system using Ink
- Inventory system
- Multiple item types
- Quest system using Reactors
- Enemy AI with NavMesh Pathfinding
- NPC patrol system
- Slime enemies with combat mechanics
- Large slimes split into smaller slimes after death
- Item collection and quest item delivery
- FMOD sound integration
- Ambient zones with different music
- Teleport system between areas

---

# Walkthrough

1. Start the game at the lake and talk to the fisherman.
2. Go to the village and talk to Kopatych.
3. Learn that the fisherman’s lucky fish was released back into the lake.
4. Return to the pier and catch the green fish.
5. Bring the fish back to the fisherman.
6. Receive the **Lucky Fish Token** as a reward.
7. Talk to Kopatych again.
8. Collect **4 radioactive mushrooms**.
9. Return to Kopatych and receive the **House Key**.
10. Enter Kopatych's house and use the teleport.

# Known Limitations

Due to time constraints, the final environments were not fully completed.

After entering Kopatych's house, the player first arrives on a small island. This island was originally intended to represent the interior of the house.

Walking to the right teleports the player to another small island. This island was planned to become the final waterfall cave area.

The complete house interior and cave environments were not finished before the submission deadline, but the teleport logic and gameplay progression are fully functional.
